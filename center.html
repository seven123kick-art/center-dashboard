クラウド接続設定済み（自動）
開くたびに自動でクラウドと同期します。追加設定は不要です。
📋 Supabase 設定手順（初回のみ・約10分）
1. supabase.com にアクセス → 「Start your project」→ Googleアカウントでログイン
2. 「New project」→ プロジェクト名を入力（例: kitasaitama-center）→ Tokyoリージョン選択 → パスワードを設定 → 「Create new project」
3. 左メニュー「Storage」→ 「New bucket」→ 名前: center-data → Public をOFF → 「Create bucket」
4. 左メニュー「Project Settings」→「API」→
　　Project URL をコピー → 上の「Supabase URL」に貼り付け
　　anon public キーをコピー → 上の「Anon Key」に貼り付け
5. 「接続する」ボタンを押して「✅ 接続OK」と出ればOK

2台目のPCでの設定：
同じURLとAnon Keyを入力して「接続する」→ 自動でデータが同期されます
